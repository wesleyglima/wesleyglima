## 📚 Sobre mim

Desenvolvedor com experiência no desenvolvimento de aplicativos mobile utilizando **React Native**. Domino tecnologias como **JavaScript** e **TypeScript**, além de práticas avançadas de estilização com **Styled-components** e gerenciamento de estado global com **Context API**.

Tenho experiência na implementação de estratégias **Offline First** utilizando **RealmDB**, garantindo uma experiência fluida e consistente para os usuários, mesmo sem conexão com a internet.

Sou um profissional curioso, com facilidade para aprender novas tecnologias e me adaptar a diferentes desafios.  

Acredito que minha paixão por aprender e buscar soluções inovadoras me torna um desenvolvedor mais eficiente, sempre em busca das **melhores práticas e ferramentas** para entregar projetos de alta qualidade.

## 💼 Experiência e conhecimentos

Minha experiência inclui:

- Consumo de **APIs RESTful** para integração com back-end  
- Criação de **Deep Linking** para navegação otimizada  
- Desenvolvimento de **aplicativos white label** com diferentes **flavors para Android e iOS**  
- Configuração de **Push Notifications com OneSignal**  
- **Conexão de aplicativos via Bluetooth**  
- Integração com **Firebase** (autenticação, banco de dados em tempo real e armazenamento)  
- Utilização de bancos de dados **SQL e NoSQL**  
- **Versionamento de código com Git e GitHub**

Também possuo conhecimentos em **desenvolvimento de APIs REST com Node.js**, aplicando:

- Arquitetura **Controller, Service e Repository**
- ORMs como **Sequelize e Prisma**
- Autenticação com **JWT (jsonwebtoken)**
- Upload e manipulação de arquivos com **Multer**
