## 📚 Sobre mim

Desenvolvedor com experiência no desenvolvimento de aplicativos mobile utilizando React Native. Domino tecnologias como JavaScript e TypeScript, além de práticas avançadas de estilização com Styled-components e gerenciamento de estado global com Context API.

Tenho experiência na implementação de estratégias Offline First utilizando RealmDB e SQLite, garantindo uma experiência fluida e consistente para os usuários, mesmo sem conexão com a internet.

Sou um profissional curioso, com facilidade para aprender novas tecnologias e me adaptar a diferentes desafios.  

Acredito que minha paixão por aprender e buscar soluções inovadoras me torna um desenvolvedor mais eficiente, sempre em busca das melhores práticas e ferramentas para entregar projetos de alta qualidade.

- Upload e manipulação de arquivos com Multer
